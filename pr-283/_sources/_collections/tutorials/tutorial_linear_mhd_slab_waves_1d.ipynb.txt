    "# Verification of MHD Wave Dispersion Relations\n",
    "\n",
    "## 1D Slab Waves in a Homogeneous Magnetized Plasma\n",
    "\n",
    "This tutorial demonstrates verification of the dispersion relations for the shear Alfvén, slow magnetosonic, and fast magnetosonic waves in a homogeneous magnetized plasma slab using the LinearMHD model.\n",
    "\n",
    "### Physical Setup\n",
    "\n",
    "We consider a one-dimensional equilibrium with constant density $n_0$, pressure $p_0$, and a uniform background magnetic field\n",
    "\n",
    "$$\\mathbf{B}_0 = (B_{0x}, B_{0y}, B_{0z}).$$\n",
    "\n",
    "Small-amplitude perturbations are launched with wave vector $\\mathbf{k} = k\\,\\hat{\\mathbf{z}}$, so the three linear MHD branches reduce to the standard slab-wave dispersion relations. In Struphy units, the Alfvén and sound speeds are\n",
    "\n",
    "$$v_A^2 = \\frac{|\\mathbf{B}_0|^2}{n_0}, \\qquad c_S^2 = \\gamma\\,\\frac{p_0}{n_0}.$$\n",
    "\n",
    "The three wave branches are:\n",
    "\n",
    "$$\\text{shear Alfv\\'en}: \\qquad \\omega = v_A\\,k\\,\\frac{B_{0z}}{|\\mathbf{B}_0|},$$\n",
    "\n",
    "$$\\text{slow magnetosonic}: \\qquad \\omega = k\\sqrt{\\frac{1}{2}(c_S^2 + v_A^2)\\left(1 - \\sqrt{1 - \\delta}\\right)},$$\n",
    "\n",
    "$$\\text{fast magnetosonic}: \\qquad \\omega = k\\sqrt{\\frac{1}{2}(c_S^2 + v_A^2)\\left(1 + \\sqrt{1 - \\delta}\\right)},$$\n",
    "\n",
    "with\n",
    "\n",
    "$$\\delta = \\frac{4 B_{0z}^2 c_S^2 v_A^2}{(c_S^2 + v_A^2)^2 |\\mathbf{B}_0|^2}.$$\n",
    "\n",
    "We verify these wave speeds by:\n",
    "1. Initializing small velocity perturbations (noise)\n",
    "2. Running a transient simulation\n",
    "3. Extracting dominant wave frequencies via FFT power spectrum\n",
    "4. Comparing fitted wave speeds against analytical predictions"
   ]
  },

    "### Model and Equilibrium Parameters\n",
    "\n",
    "Define the magnetic field configuration, plasma density, and pressure (via plasma beta) that sets up the equilibrium. These parameters determine the wave speeds we will verify."
   ]
  },

   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "# Magnetic field components\n",
    "B0x = 0.0\n",
    "B0y = 1.0\n",
    "B0z = 1.0\n",
    "\n",
    "# Plasma parameters\n",
    "beta = 3.0  # ratio of thermal to magnetic pressure\n",
    "n0 = 0.7    # reference density\n",
    "\n",
    "# Compute thermal pressure from beta and magnetic pressure\n",
    "Bsquare = B0x**2 + B0y**2 + B0z**2\n",
    "p0 = beta * Bsquare / 2.0\n",
    "\n",
    "print(f\"Magnetic field: B0 = ({B0x}, {B0y}, {B0z})\")\n",
    "print(f\"Alfvén speed: v_A = {np.sqrt(Bsquare / n0):.4f}\")\n",
    "print(f\"Sound speed: c_s = {np.sqrt(5/3 * p0 / n0):.4f}\")\n",
    "print(f\"Beta (plasma): {beta}\")"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "### Domain and Numerical Discretization\n",
    "\n",
    "Set up a 1D periodic domain in the $z$-direction with a tensor-product grid. Since the wave propagates along $z$, we keep $x$ and $y$ as single-element."

    "grid = grids.TensorProductGrid(num_elements=(1, 1, 64))\n",
    "\n",
    "# Derham options for 1D:\n",
    "# - degree=(1,1,3): cubic edges in z for smooth wave representation\n",
    "derham_opts = DerhamOptions(degree=(1, 1, 3))\n",
    "\n",
    "print(f\"Domain: z ∈ [0, {domain.params['r3']})\")\n",
    "print(f\"Grid elements: {grid.num_elements}\")\n",

    "model.mhd.velocity.add_perturbation(perturbations.Noise(amp=0.1, comp=2, seed=123))\n",
    "\n",
    "# Build equilibrium object\n",

    "### Diagnostics: FFT Analysis and Wave Speed Verification\n",
    "\n",
    "Extract velocity and pressure data over time, compute power spectra, and fit dominant frequencies to extract wave speeds. We compare the fitted speeds against the theoretical predictions for shear Alfvén, slow and fast magnetosonic waves.\n",
    "\n",
    "**Shear Alfvén wave**: expected speed $\\approx {0.707} \\, v_A$\n",
    "\n",
    "**Slow and fast magnetosonic waves**: derived from dispersion relation involving both $v_A$ and $c_s$."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "# Load plotting data\n",
    "sim.load_plotting_data()\n",
    "\n",
    "# Extract velocity and pressure time-series\n",
    "u_of_t = sim.spline_values.mhd.velocity_log.data\n",
    "p_of_t = sim.spline_values.mhd.pressure_log.data\n",
    "\n",
    "# Dispersion relation parameters\n",
    "gamma = 5 / 3  # Adiabatic index\n",
    "disp_params = {\n",
    "    \"B0x\": B0x,\n",
    "    \"B0y\": B0y,\n",
    "    \"B0z\": B0z,\n",
    "    \"p0\": p0,\n",
    "    \"n0\": n0,\n",
    "    \"gamma\": gamma,\n",
    "}\n",
    "\n",
    "# 1. Shear Alfvén wave analysis from velocity\n",
    "print(\"\\n=== Shear Alfvén Wave Analysis ===\")\n",
    "_1, _2, _3, coeffs_alfven = power_spectrum_2d(\n",
    "    u_of_t,\n",
    "    \"velocity_log\",\n",
    "    grids=sim.grids_log,\n",
    "    grids_mapped=sim.grids_phy,\n",
    "    component=0,\n",
    "    slice_at=[0, 0, None],\n",
    "    do_plot=True,\n",
    "    disp_name=\"MHDhomogenSlab\",\n",
    "    disp_params=disp_params,\n",
    "    fit_branches=1,\n",
    "    noise_level=0.5,\n",
    "    extr_order=10,\n",
    "    fit_degree=(1,),\n",
    ")\n",
    "\n",
    "# Theoretical Alfvén speed\n",
    "vA = xp.sqrt(Bsquare / n0)\n",
    "v_alfven_theory = vA * B0z / xp.sqrt(Bsquare)\n",
    "v_alfven_fit = float(coeffs_alfven[0][0])\n",
    "\n",
    "print(f\"Théoretical Alfvén speed: {v_alfven_theory:.6f}\")\n",
    "print(f\"Fitted Alfvén speed:      {v_alfven_fit:.6f}\")\n",
    "print(f\"Relative error:           {abs(v_alfven_fit - v_alfven_theory) / v_alfven_theory * 100:.2f}%\")\n",
    "\n",
    "error_alfven = xp.abs(coeffs_alfven[0][0] - v_alfven_theory)\n",
    "assert error_alfven < 0.07, f\"Alfvén wave speed error {error_alfven:.4f} exceeds tolerance\"\n",
    "print(\"✓ Alfvén wave verification passed.\\n\")"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "# 2. Magnetosonic waves analysis from pressure\n",
    "print(\"=== Slow and Fast Magnetosonic Wave Analysis ===\")\n",
    "_1, _2, _3, coeffs_sonic = power_spectrum_2d(\n",
    "    p_of_t,\n",
    "    \"pressure_log\",\n",
    "    grids=sim.grids_log,\n",
    "    grids_mapped=sim.grids_phy,\n",
    "    component=0,\n",
    "    slice_at=[0, 0, None],\n",
    "    do_plot=True,\n",
    "    disp_name=\"MHDhomogenSlab\",\n",
    "    disp_params=disp_params,\n",
    "    fit_branches=2,\n",
    "    noise_level=0.4,\n",
    "    extr_order=10,\n",
    "    fit_degree=(1, 1),\n",
    ")\n",
    "\n",
    "# Theoretical magnetosonic speeds\n",
    "cS = xp.sqrt(gamma * p0 / n0)\n",
    "delta = (4 * B0z**2 * cS**2 * vA**2) / ((cS**2 + vA**2) ** 2 * Bsquare)\n",
    "v_slow_theory = xp.sqrt(0.5 * (cS**2 + vA**2) * (1.0 - xp.sqrt(1.0 - delta)))\n",
    "v_fast_theory = xp.sqrt(0.5 * (cS**2 + vA**2) * (1.0 + xp.sqrt(1.0 - delta)))\n",
    "\n",
    "v_slow_fit = float(coeffs_sonic[0][0])\n",
    "v_fast_fit = float(coeffs_sonic[1][0])\n",
    "\n",
    "print(\"\\nSlow Magnetosonic Wave:\")\n",
    "print(f\"  Théoretical speed: {v_slow_theory:.6f}\")\n",
    "print(f\"  Fitted speed:      {v_slow_fit:.6f}\")\n",
    "print(f\"  Relative error:    {abs(v_slow_fit - v_slow_theory) / v_slow_theory * 100:.2f}%\")\n",
    "\n",
    "print(\"\\nFast Magnetosonic Wave:\")\n",
    "print(f\"  Théoretical speed: {v_fast_theory:.6f}\")\n",
    "print(f\"  Fitted speed:      {v_fast_fit:.6f}\")\n",
    "print(f\"  Relative error:    {abs(v_fast_fit - v_fast_theory) / v_fast_theory * 100:.2f}%\")\n",
    "\n",
    "error_slow = xp.abs(coeffs_sonic[0][0] - v_slow_theory)\n",
    "error_fast = xp.abs(coeffs_sonic[1][0] - v_fast_theory)\n",
    "\n",
    "assert error_slow < 0.05, f\"Slow wave speed error {error_slow:.4f} exceeds tolerance\"\n",
    "assert error_fast < 0.19, f\"Fast wave speed error {error_fast:.4f} exceeds tolerance\"\n",
    "print(\"\\n✓ Magnetosonic wave verification passed.\")"

    "### Conclusion\n",
    "\n",
    "This tutorial successfully verified the dispersion relations for three branches of MHD waves in a homogeneous magnetized plasma:\n",
    "\n",
    "1. **Shear Alfvén wave** propagates perpendicular to the ambient magnetic field at the magnetic tension-driven speed.\n",
    "2. **Slow magnetosonic wave** is a hybrid mode combining acoustic and magnetic effects, with lower phase velocity.\n",
    "3. **Fast magnetosonic wave** is a magnetically-dominated mode with higher phase velocity, approaching $\\sqrt{c_s^2 + v_A^2}$ at high frequencies.\n",
    "\n",
    "The FFT-based power spectrum analysis provides a robust method to extract wave speeds from simulation data and verify the model's fidelity to the underlying MHD theory. This verification validates both the physical model implementation and the numerical discretization."
