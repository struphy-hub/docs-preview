    "# Kinetic Plasma Dynamics\n",
    "\n",
    "## Weak Landau Damping in Vlasov-Ampère Model\n",
    "\n",
    "This tutorial demonstrates verification of the kinetic Vlasov-Ampère solver using the weak Landau damping phenomenon. The `VlasovAmpereOneSpecies` model simulates the evolution of particle distribution functions coupled to self-consistent electromagnetic fields.\n",
    "\n",
    "### Physical Setup\n",
    "\n",
    "Landau damping is a collisionless plasma phenomenon where small-amplitude Langmuir waves are damped through resonant interaction with the particle distribution. In the weak damping regime, a sinusoidal electric field perturbation decays exponentially over time:\n",
    "\n",
    "$$E(t) \\propto \\exp(2\\gamma t),$$\n",
    "\n",
    "where $\\gamma = -0.1533$ is the analytical damping rate (for specific initial conditions and plasma parameters).\n",
    "\n",
    "We verify this damping by:\n",
    "1. Initializing a Maxwellian distribution with a sinusoidal density perturbation\n",
    "2. Running a long kinetic simulation (t=0 to 15)\n",
    "3. Monitoring the electric field energy over time\n",
    "4. Extracting the damping rate via envelope fitting to wave peaks\n",
    "5. Comparing against the theoretical rate $\\gamma = -0.1533$"
   ]

    "### Damping Rate Analysis: Peak Envelope Fitting\n",
    "\n",
    "Find local maxima in the log-energy curve and fit a linear trend to extract the exponential damping rate. The slope of the fitted line is $2\\gamma$."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "17",
   "metadata": {},
   "outputs": [],
   "source": [
    "# Find peaks (local maxima) by checking sign changes in derivative\n",
    "dEdt = (xp.roll(logE, -1) - xp.roll(logE, 1))[1:-1] / (2.0 * time_opts.dt)\n",
    "zeros = dEdt * xp.roll(dEdt, -1) < 0.0  # Sign change indicates extremum\n",
    "maxima_inds = xp.logical_and(zeros, dEdt > 0.0)  # Positive slope before zero = maximum\n",
    "\n",
    "maxima = logE[1:-1][maxima_inds]\n",
    "t_maxima = time[1:-1][maxima_inds]\n",
    "\n",
    "print(f\"\\nFound {len(t_maxima)} local maxima in the oscillating signal.\")\n",
    "print(f\"First 5 peak times: {t_maxima[:5]}\")\n",
    "print(f\"First 5 peak energies (log10): {maxima[:5]}\")"
   ]
  },

    "### Damping Rate Calculation\n",
    "\n",
    "Perform a linear least-squares fit to the first five peak maxima to extract the damping rate. Compare the fitted rate against the theoretical value $\\gamma = -0.1533$."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "19",
   "metadata": {},
   "outputs": [],
   "source": [
    "# Use first 5 peaks for damping fit (linear regime)\n",
    "n_peaks = min(5, len(t_maxima))\n",
    "t_fit = t_maxima[:n_peaks]\n",
    "e_fit = maxima[:n_peaks]\n",
    "\n",
    "# Linear regression: log(E) = 2*gamma*t + const\n",
    "# Slope = 2*gamma, so gamma = slope / 2\n",
    "linfit = xp.polyfit(t_fit, e_fit, 1)\n",
    "slope = float(linfit[0])\n",
    "gamma_fit = slope / 2.0\n",
    "\n",
    "# Theoretical Landau damping rate\n",
    "gamma_theory = -0.1533\n",
    "\n",
    "print(\"\\n=== Weak Landau Damping Analysis ===\")\n",
    "print(f\"\\nLinear fit to {n_peaks} peak maxima:\")\n",
    "print(f\"  Slope (d(log E)/dt):  {slope:.6f}\")\n",
    "print(f\"  Damping rate (γ_fit): {gamma_fit:.6f}\")\n",
    "print(f\"  Theoretical (γ_th):   {gamma_theory:.6f}\")\n",
    "print(f\"  Relative error:       {xp.abs(gamma_fit - gamma_theory) / xp.abs(gamma_theory) * 100:.2f}%\")"
   ]
